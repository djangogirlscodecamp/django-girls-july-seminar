HTML

HTML은 무엇인가요?

HTML은 HyperText Markup Language의 약자입니다.

HyperText

HyperText는 한 문서에서 다른 문서로 이동이 가능한 텍스트를 의미합니다.

Markup Language

Markup Language 문서나 데이터의 구조를 명기하는 언어입니다.

HTML의 기본 구조

doctype

<!doctype html>와 같은 형태로 최상단에 선언 되는 doctype은 여러가지 html 버전이 있기 때문에, 어떤 버전의 HTML을 사용할 것인지 명시해줍니다.

html

<html> ... </html> 와 같은 형태로 문서를 감싸줍니다.

head

<head> ... </head> html 태그와 마찬가지로 이런형식으로 감싸집니다. 기본적으로 html의 대부분 태그는 이런식으로 감싸진다는 걸 기억하세요(예외도 존재합니다.) head 태그는 보통 컴퓨터에게 알려주는 정보들을 명기합니다.

meta

meta 태그는 작성되고 있는 HTML 문서 자체의 정보들을 명기합니다.

style

style 태그 안에는 보통 css와 같은 스타일시트의 내용이 들어옵니다.

<style type="text/css">

/* 주석은 요렇게! */

body {

background-color: rgba(155, 155, 0, 0.2);

}

</style>

link

link 태그는 외부 자원을 연결해주는 데 사용됩니다. 스타일시트를 링크하는데 많이 사용합니다.

<link rel="stylesheet" href="/css/djangogirls.css" charset="utf-8">

script

스크립트 코드를 적기 위해 사용합니다.

<script> alert("안녕하세요! 만나서 반갑습니다."); </script>

보통은 아래와 같이 외부의 코드를 불러와서 사용합니다.

<script src="https://code.jquery.com/jquery-3.1.0.slim.min.js" ></script>

body

보통 실제 유저에게 보여지는 정보들을 적어줍니다. 지금 여러분이 보시고 있는 화면도 body 태그의 내용이에요!

div

div 태그는 가장 많이 쓰이는 태그 중 하나입니다. division의 약자이고, html 내에서 구역(?)을 구분할 때 쓰입니다.

heading(h1 ... h6)

heading 태그는 보통 제목을 표시할 때 많이 사용합니다. <h1>와 같은 형태로 표현하며, 아래와 같이 6가지로 표현됩니다.

p

p 태그는 paragraph의 약자로 문락 또는 단락을 나눌 때 사용됩니다.
